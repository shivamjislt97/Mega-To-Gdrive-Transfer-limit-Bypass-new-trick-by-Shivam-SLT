📁 MEGA LINKS Secret Generator

GitHub Actions workflow ke liye MEGA_LINKS secret JSON banayein. Ek click mein copy karein.

📖 User Guide — Step by Step
1
Folder name daalein (e.g. Movies) Enter the folder name — yehi GDrive mein folder banega
2
MEGA links paste karein — har line par ek link Paste each MEGA file URL on a separate line
3
"+ Add Folder" button click karein — folder list mein add ho jayega Click to add the folder to the list
4
Saare folders isi tarah add karein (jitne chahein) Repeat for all folders you want to include
5
"🚀 Generate Secret" click karein — minified JSON ban jayega Click to generate the single-line JSON secret
6
📋 Copy karo → GitHub repo → Settings → Secrets and variables → MEGA_LINKS mein paste karo Copy the output and paste it as a GitHub Actions secret
7
🔄 Reset State — upar Reset State section se {"folders": {}} copy karo, completed_links.json mein paste karo Copy the reset JSON from the Reset State section above to clear transfer progress
